Elements

In Monster Legends there are 10 elements. Each different element is strong against another. The 10 different elements are fire, nature, earth, thunder, water, dark, magic, light, legendary and metal. All these different elements also have a different range of powers. There are no good or bad elements but the element that does the most damage is the legendary element.

Legendary Element

The legendary element is more of a category than an element. It is a breed of monsters that have the opportunity to reach a higher helth points level than some othe lower rariety monsters. This naturally points them in a higher ranked position. The legendary element is also less breedable meaning thare are less known combiinations you can use to breed a few legendary monsters. Fire Element

The fire element is the element that the first character you receive is. This element is also has alot more powers that can aply burning and bleeding. These attacks also do more damage than other elements apply. This gives a huge advantage when this element verses the nature element.

Nature Element

The nature element is the element of the second character you receive is. This element focuses on stunning and poisoning there enemies. This is useful when versing the magic element and in a tough situation.

Earth Element

The earth element is an element which focuses on protecting there alles and aplying shields. Earth characters are very useful when battling in the adventure map against hard characters.

Thunder Element

The thunder element is an element that has an advantage over water. This element uses the legendary powers instead of the thunder powers but when used pacs a punch to the enemie.

Water Element

The water element is useful against the fire element. This element can provide damage boost and other effects. A useful element for providing more strength fo your best monsters.

Dark Element

The dark element is an element that is good at applying negative effects and doing a great deal of damage. This is useful against earth and is a good element to have on you team.

Magic Element

The magic element is an element that focuses on applying random negative effects to there enemy and loves confusing there opponent. This element is strong against light so watch out if you are versing the magic element and enjoy this element when used because magic characters pack a punch.
